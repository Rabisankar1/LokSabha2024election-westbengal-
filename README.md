# 🇮🇳 Election 2024 Data Analysis

## 📌 Overview
This project analyzes candidate-level data from the 2024 Indian General Election to uncover voting patterns, party performance, and electoral competitiveness.

## 📊 Features
- Party-wise vote analysis
- Vote share distribution
- Winning margin insights
- Gender participation analysis
- Top candidate performance

## 🛠️ Tech Stack
- Python
- Pandas
- Seaborn
- Matplotlib

## 📈 Key Insights
- Major parties dominate vote share
- Close contests observed in many constituencies
- Gender gap in political participation
- High vote concentration among top candidates
